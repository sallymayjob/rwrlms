# Launch Guide

## Goal
Deploy a complete Slack-native LMS without external infrastructure.

## Prerequisites

- Google Workspace account with Apps Script + Sheets access
- Slack workspace with app creation permissions
- CSV content generated by LMS Composer

## Launch Checklist

1. Create `RWR_LMS_DATABASE` sheet and tabs.
2. Import CSV templates from `/database`.
3. Configure Apps Script project from `/apps-script` files.
4. Set Script Properties for secrets and IDs.
5. Deploy Web App endpoint.
6. Create Slack app from manifest and install to workspace.
7. Set slash command endpoints.
8. Validate command flow with pilot users.

## Pilot Validation Script

- `/onboard`
- `/courses`
- `/enroll C001`
- `/learn`
- `/submit M03-W02-L04 complete`
- `/progress`

Expected result: profile created, enrollment succeeds, lesson delivered, submission tracked, progress increased.
